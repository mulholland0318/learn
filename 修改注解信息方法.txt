修改提交时的注释信息
我们对某一次提交注释了信息，提交完成后发现，注释信息不对或者注释信息不能完全体现本次提交的变动内容，
这是就要对注释信息进行修改，修改注释信息分两种情况。
1.修改最近一次提交的注释信息，
这种情况修改提交时的注释信息比较简单通过如下命令完成。
git commit --amend
运行上面命令后，就打开的最新一次提交的注释信息的编辑界面。
编辑完成后保存，最新一次提交的注释信息就修改完成了。

2.修改最新一次提交之前的一次提交注释信息。
首先 通过如下命令选择修改哪次提交的注释信息。
git rebase -i HEAD~3   (该命令表示修改当前版本倒数第三次的状态)

执行命令后  格式为
pick: ********
pick: *********
pick: *********
如果要修改哪个 ，就把pick改成edit ，然后保存推出。
这时候最新一次提交就是 把pick变成edit 的那次提交了，
然后使用
之前使用的命令 git commit  --amend
修改信息如第一种情况一样.
编辑完成后 执行如下命令
git rebase --continue  
命令执行完成后 相应提交的注释信息就修改完成了
